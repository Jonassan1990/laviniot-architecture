## Breakpoints

| Name | Width | Target |
|---|---|---|
| Mobile | `< 640px` | Smartphones |
| Tablet | `640px – 1024px` | Tablets, industrial HMIs |
| Desktop | `> 1024px` | Workstations, control room screens |

## Mobile priorities

- Critical alarm notifications must be actionable from mobile
- Overview and device status screens must be fully functional at mobile width
- Heavy data entry (device configuration) is desktop-first; mobile is read-only for these screens
