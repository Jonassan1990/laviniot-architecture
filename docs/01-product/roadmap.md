# Roadmap

:::note Living document
The roadmap reflects current priorities. It is updated as decisions are made. It is not a commitment to specific delivery dates.
:::

---

## Phase 1 — Foundation

**Goal:** A deployable platform that receives telemetry from real devices and surfaces it in a dashboard.

| Milestone | Status |
|---|---|
| Marketing website live at `www.lavin-iot.com` | ✅ Complete |
| Architecture portal (Lavin Book) initialised | ✅ Complete |
| ADR framework established | ✅ Complete |
| Engineering Principles documented | ✅ Complete |
| API contract defined (v1) | 🟡 In Progress |
| API backend scaffold (Flask) | 🟡 In Progress |
| Platform dashboard scaffold (Next.js) | 🟡 In Progress |
| MQTT ingestion | ⬜ Planned |
| OPC-UA provider | ⬜ Planned |
| Device registry | ⬜ Planned |
| Telemetry storage (InfluxDB) | ⬜ Planned |
| Basic alarm rules (threshold) | ⬜ Planned |
| Email notifications | ⬜ Planned |
