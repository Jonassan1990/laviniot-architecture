## Colour tokens

| Token | Value | Usage |
|---|---|---|
| `--color-primary` | `#0B6E4F` | Primary actions, links, focus rings |
| `--color-primary-dark` | `#074a34` | Hover states |
| `--color-alarm-critical` | `#DC2626` | Critical alarm indicators |
| `--color-alarm-warning` | `#D97706` | Warning alarm indicators |
| `--color-alarm-info` | `#2563EB` | Info alarm indicators |
| `--color-alarm-ok` | `#16A34A` | Resolved / healthy state |
| `--color-surface` | `#FFFFFF` | Card backgrounds |
| `--color-background` | `#F9FAFB` | Page background |
| `--color-text-primary` | `#111827` | Primary text |
| `--color-text-muted` | `#6B7280` | Secondary text |

## Typography

| Token | Value |
|---|---|
| `--font-family` | Inter, system-ui, sans-serif |
| `--font-size-base` | 14px (dashboard) / 16px (marketing) |
| `--font-weight-normal` | 400 |
| `--font-weight-medium` | 500 |
| `--font-weight-bold` | 600 |

## Spacing scale

Uses a 4px base unit: `4, 8, 12, 16, 24, 32, 48, 64, 96`.
